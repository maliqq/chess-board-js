# Chess Board JS

Lightweight chess board viewer with PGN playback, opening explorer, and shareable FEN links.

Demo: https://chess-board-js.vercel.app/

## Features
- Interactive board with legal move highlighting
- PGN playback and FEN editing
- Opening explorer with search
- Shareable URL synced to current FEN

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
